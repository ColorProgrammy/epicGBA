GBA PROJECT: MYGBAPROJECT3
=============================

Created with SimpleGBA-IDE

QUICK START:
1. Double-click build.bat to compile
2. Double-click play.bat to compile and run in mGBA

FILES:
- build.bat      - Main build script (uses Makefile)
- play.bat       - Build and run in emulator
- Makefile       - Build configuration
- source/main.c  - Source code
- include/gba.h  - GBA header

CONTROLS:
- A button: Change to yellow
- B button: Change to purple

DEVELOPMENT:
1. Edit source/main.c to change game logic
2. Add graphics to graphics/ folder
3. Add sounds to sound/ folder

REQUIREMENTS:
- DevkitPro installed at C:\devkitPro
- mGBA emulator for testing (install to default path)
